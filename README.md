# Portfolio Pro - Full Admin System

## Frontend
```bash
cd frontend
npm install
npm run dev
```

## Backend
```bash
cd backend
python -m venv .venv
# activate venv
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## URLs
- Frontend: http://localhost:3000
- Backend: http://127.0.0.1:8000
- Swagger: http://127.0.0.1:8000/docs
